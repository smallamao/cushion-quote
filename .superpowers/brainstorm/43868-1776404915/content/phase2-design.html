2-9. 列表頁面統一卡片化

每個列表頁面在 ≤768px 時從表格切換為卡片列表

報價紀錄
卡片欄位：單號 + 狀態、客戶名、日期、金額
操作：編輯、複製 icon 按鈕
篩選：搜尋框 + 狀態下拉（取代 inline select）
案件紀錄
卡片欄位：案件名 + 狀態、客戶、來源、建立日期
操作：點擊展開詳情（報價/採購/成本）
篩選：搜尋框 + 狀態下拉
採購單列表
卡片欄位：單號 + 狀態、廠商、日期、金額
操作：點擊進入詳情頁
篩選：搜尋框 + 狀態 pill 改為水平捲動
採購單編輯器
表單：供應商欄位垂直堆疊
品項：卡片式（同報價編輯器模式）
ProductCombobox：全寬下拉
材質資料庫
卡片欄位：品牌/系列、色號/色名、進價/牌價
操作：編輯、停用 icon 按鈕
隱藏：分類欄（次要資訊）
佣金結算

表格：隱藏付款日期欄
統計卡片：保持原樣（已 OK）
售後服務

分頁：簡化為 icon 按鈕
ProductCombobox

結果項：雙行排列（名稱 + 價格換行）
