  • 桌面版 sidebar 行為完全不變

改動細節：
  • Header padding：px-8 → px-4 md:px-8
  • 通知面板：w-80 → w-[calc(100vw-32px)] md:w-80，手機上接近全寬
  • 使用者選單：w-52 → w-[calc(100vw-32px)] md:w-52
  • 觸控目標：鈴鐺/頭像從 h-9 w-9 → h-10 w-10

3. 全域觸控尺寸升級

按鈕和輸入框在手機上放大到 44px 觸控區域

現在
h-8 (32px) sm 按鈕
h-9 (36px) 預設按鈕
h-8 輸入框
改善後（手機）
h-10 (40px) sm 按鈕
h-11 (44px) 預設按鈕
h-11 輸入框
策略：CSS 變數 + 媒體查詢，手機上自動放大，桌面不變。
  • button.tsx：sm size h-8 → h-10 md:h-8
  • button.tsx：default size h-9 → h-11 md:h-9
  • button.tsx：icon size h-8 w-8 → h-10 w-10 md:h-8 md:w-8
  • input.tsx：同樣邏輯，確保 ≥44px
  • page-container padding：var(--space-8) → 手機 16px / 桌面 32px

4. CSS 斷點體系

建立統一的響應式斷點，取代目前的單一 1080px

≤ 768px — 手機：sidebar 隱藏，漢堡抽屜，全域觸控放大
769px ~ 1080px — 平板：sidebar 自動收合為 icon-only (56px)
≥ 1081px — 桌面：現有行為不變
改動：
  • globals.css 新增 @media (max-width: 768px) 區塊
  • 768px 以下：.app-shell { grid-template-columns: 1fr } + .sidebar { display: none }
  • 769-1080px：保留 sidebar 但強制收合
  • .page-container 手機 padding 降為 16px
