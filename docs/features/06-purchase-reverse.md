# 06-purchase-reverse.md

## 功能概述
- 用途說明：處理採購成本與最終售價的邏輯關聯，支援成本變動時的售價自動更新。
- 使用者角色：業務人員

## 技術架構

### 核心運算
- **材料費**：`(才數 / 面料寬度) * 面料單價 * (1 + 損耗率)`。
- **總成本**：`材料費 + 工資 + 泡棉費`。
- **售價**：`總成本 * 通路倍率 + 分攤佣金`。

## 功能細節
- 當用戶在報價單上手動修改數量或成本時，系統會觸發 `recalculate` 邏輯，確保利潤空間符合預期。
